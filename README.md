# qiankun-react-example

一个可用于生产环境的基于react18+vue3+qiankun2的微前端模版qiankun 实战 dem，父应用 react18，子应用使用 `react18`, `vue3`。

如果对您有一点帮助的，麻烦给下star 🌟 。 万分感谢～～～～

开始

安装根目录工程依赖

```
npm i
```

一键安装所有主子应用的依赖

```
npm run install
```

一键启动所有所有应用

```
npm start
```

通过 [http://localhost:3000/](http://localhost:8080/) 访问主应用。

## 发布

一键构建并打包所有主子应用

```
npm run build
```
